# NOM DU PROJET 

Approche in silico pour la simulation et l’augmentation de la production des métabolites spécialisés chez Penicillium rubens. 

## DESCRIPTION 

```html
<html>
Notre projet est une application web développée avec NiceGUI permettant de charger et analyser des modèles métaboliques SBML via COBRApy. 
</html>
```

# BACKGROUND 

Ce projet avait pour but initalement d'étudier le réseau métabolique iPrub22 réalisé par Delphine Nègre mais peut être utilisé pour d'autres réseaux métaboliques. Il permet d'étudier les caractéristiques du modèle (informations diverses sur les métabolites, gènes et réactions le constituants). L'utilisateur peut ensuite réaliser des FBA et FVA avec une certaine modularité : il peut choisir de modifier les contraintes des bornes du modèle, définir une autre fonction objectif... 

## TECHNOLOGIES UTILISEES 

COBRApy est une bibliothèque Python permettant la modélisation et l'analyse de réseaux métaboliques à l'aide de contraintes (Constraint-Based Reconstruction and Analysis). Elle permet d'étudier le fonctionnement métabolique d'organismes à partir de modèles au format SBML. 

NICEGUI est une bibliothèque Python open source permettant de créer des interfaces graphique fonctionnant dans le navigateur.
